# Light Switch Runner

## Опис проєкту

**Light Switch Runner** — це 2D one-button arcade runner, створений у Unity для лабораторної роботи №6.
Гравець автоматично рухається вперед, а єдина дія гравця — перемикання режиму світу між **Light** та **Dark**.

Гра має нескінченну генерацію платформ, систему рахунку, поступове збільшення швидкості, умову програшу та можливість перезапуску.

---

## Механіка гри в одному реченні

Гравець автоматично рухається вперед, а одним натисканням перемикає світло і темряву, щоб активувати потрібні платформи та протриматися якомога довше.

---

## Керування

| Дія                           | Кнопка            |
| ----------------------------- | ----------------- |
| Перемкнути режим Light / Dark | ЛКМ / Space / Tap |
| Пауза                         | UI-кнопка Pause   |
| Продовжити гру                | UI-кнопка Resume  |
| Перезапуск                    | UI-кнопка Restart |
| Вихід у меню                  | UI-кнопка Quit    |

---

## Core Loop

1. Гравець запускає гру з головного меню.
2. Персонаж автоматично рухається вперед.
3. На шляху нескінченно генеруються платформи різної ширини.
4. Платформи належать до одного з двох режимів: **Light** або **Dark**.
5. Одним натисканням гравець перемикає поточний режим світу.
6. Активними залишаються тільки платформи поточного режиму.
7. Швидкість персонажа поступово збільшується.
8. Якщо гравець падає вниз — настає програш.
9. Після програшу гравець може перезапустити гру.

## Умова програшу

Гравець програє, якщо падає нижче ігрової зони.
Це відбувається тоді, коли гравець не встигає вчасно перемкнути режим світу і активна платформа зникає.

---

## Система прогресу

Прогрес гравця відстежується через рахунок.
Очки нараховуються за пройдену дистанцію. Чим довше гравець залишається живим, тим більший рахунок він отримує.

Також у грі зберігається найкращий результат за допомогою `PlayerPrefs`.

---

## Особливість гри

Особливість гри полягає в тому, що персонаж не стрибає і не керується напряму.
Гравець впливає лише на стан світу, перемикаючи активні платформи між світлим і темним режимом.

## Як запустити проєкт

1. Відкрити Unity Hub.
2. Додати папку проєкту через **Add project from disk**.
3. Відкрити проєкт у сумісній версії Unity.
4. Відкрити сцену `MainMenu`.
5. Натиснути **Play**.
